XPath Homework

  1. How would you isolate the ID number of this encoded text (VAA4025-109-1-a16)?

    /TEI.2//text/body//div[1]/@id

  2. How many lists are in the article text?

    There is 1 list. Found with the XPath: count(//list)

  3. How would you isolate the copyright statement (Copyright 2013 Trustees of Indiana University)?

    /TEI.2/teiHeader/fileDesc/publicationStmt/availability/p[1]

  4. What are two different ways to isolate the publisher and pubPlace elements together (Indiana University Libraries Bloomington, IN)?

    /TEI.2/teiHeader/fileDesc/publicationStmt/*[position() < 3]

    /TEI.2/teiHeader/fileDesc/publicationStmt/publisher,pubPlace

  5. How many paragraphs are in the text of this article?

    There are 34 paragraphs in the text of the article.

    count(/TEI.2/text//p)

  6. What different states are mentioned in the text?

    Indiana, Ohio, South Carolina, Kentucky, Arkansas, North Carolina. Plus "Monroe" which is miscoded as a state.

    Retrieved with the XPath: distinct-values(//region[@type="state"]/@reg)

  7. How many distinct cities are mentioned in the text? Hint: use distinct-values()

    25 cities. distinct-values(//settlement[@type="city"])

  8. How many times does "Evansville" appear in this article? Hint: use ctrl+F to check your work

    Evansville appears twice. Retrieved with the XPath: //settlement[@reg="Evansville"]

  9. What's your best guess for how to figure out how many book reviews there are in this article using XPath alone?

    count(//p/hi[@rend="i"]/following-sibling::lb[1])

    Find the with @rend="i" for which the following sibling is an tag, then it finds only the first sibling in order to count the title only once. Then, count them all.

    This XPATH retrieves 18 results.
